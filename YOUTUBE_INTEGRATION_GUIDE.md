- РюЁ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў YouTube upload рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА

- РюЁ рЃЋрЃўрЃЊрЃћрЃЮрЃА embed рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЕрЃљрЃАрЃЏрЃљ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃерЃў

- РюЁ Error handling рЃЊрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў

## РџЎ№ИЈ Backend рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### 1. Environment рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃћрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў `server/.env` рЃцрЃљрЃўрЃџрЃерЃў:

```env

YOUTUBE_CLIENT_ID=123456789.apps.googleusercontent.com
YOUTUBE_CLIENT_SECRET=GOCSPX-your-client-secret
YOUTUBE_REDIRECT_URI=http://localhost:4000/v1/youtube/oauth2callback
YOUTUBE_PLAYLIST_ID=PLxxxxxxxxxxxxxxxxxx
YOUTUBE_REFRESH_TOKEN=  # рЃћрЃА рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃюрЃљрЃЉрЃўрЃ»рЃерЃў

### 2. рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊрЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ (Refresh Token-рЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ)

рЃћрЃА:
1. рЃњрЃљрЃ«рЃАрЃюрЃўрЃА browser-рЃА authorization URL-рЃўрЃЌ

3. рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃфрЃћрЃЏрЃљ
4. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃћрЃюрЃА Refresh Token-рЃА

### рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЋрЃўрЃЊрЃћрЃЮрЃЌрЃў

1. **рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ Admin Panel-рЃерЃў**: `/admin/products`

3. **рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ**

   - рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃАрЃљрЃ«рЃљрЃЋрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃА рЃЊрЃљ рЃќрЃЮрЃЏрЃљрЃА

```

### рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃЋрЃўрЃЊрЃћрЃЮрЃА рЃюрЃљрЃ«рЃЋрЃљ

рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃа рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ:

- рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃюрЃљрЃ«рЃЋрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃАрЃљрЃўрЃбрЃќрЃћ

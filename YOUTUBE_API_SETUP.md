- **App name**: `FishHunt Product Manager`
- **User support email**: рЃЌрЃЦрЃЋрЃћрЃюрЃў email

4. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "SAVE AND CONTINUE"

### 3.3 Test Users рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "ADD USERS"

3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "SAVE AND CONTINUE"

### 3.4 OAuth Client ID-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
1. рЃЏрЃљрЃарЃфрЃ«рЃћрЃюрЃљ рЃЏрЃћрЃюрЃўрЃБрЃЊрЃљрЃю: "APIs & Services" Рєњ "Credentials"
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "CREATE CREDENTIALS" Рєњ "OAuth client ID"
3. **Application type**: рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ "Web application"
4. **Name**: `FishHunt Web Client`
5. **Authorized JavaScript origins**: рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ:
   - `http://localhost:3000`
   - `http://localhost:3001`
   - рЃЌрЃЦрЃЋрЃћрЃюрЃў production domain (рЃЏрЃљрЃњ: `https://fishhunt.ge`)
6. **Authorized redirect URIs**: рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ:
   - `http://localhost:3000/api/youtube/oauth2callback`
   - `http://localhost:3001/api/youtube/oauth2callback`
   - `https://yourdomain.com/api/youtube/oauth2callback`
7. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "CREATE"

### 3.5 Credentials-рЃўрЃА рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃљ
1. рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ modal window Client ID рЃЊрЃљ Client Secret-рЃўрЃЌ

4. **рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў**: рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃБрЃюрЃЊрЃљ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЮрЃЌ Git-рЃерЃў!

## рЃюрЃљрЃЉрЃўрЃ»рЃў 4: Service Account-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ (рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃў)

## рЃюрЃљрЃЉрЃўрЃ»рЃў 6: Environment рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

### Server-рЃўрЃА .env рЃцрЃљрЃўрЃџрЃерЃў (.env or .env.local):

```env
# YouTube API Configuration
YOUTUBE_CLIENT_ID=your-client-id.apps.googleusercontent.com
YOUTUBE_CLIENT_SECRET=your-client-secret
YOUTUBE_REDIRECT_URI=http://localhost:3001/api/youtube/oauth2callback
YOUTUBE_PLAYLIST_ID=PLxxxxxxxxxxxxxxxxxx
YOUTUBE_REFRESH_TOKEN=   # рЃћрЃА рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ
```

### рЃарЃЮрЃњрЃЮрЃа рЃЏрЃЮрЃЋрЃўрЃърЃЮрЃЋрЃЮрЃЌ Client ID рЃЊрЃљ Secret?
1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃўрЃџрЃў `youtube-credentials.json` рЃцрЃљрЃўрЃџрЃў
2. рЃўрЃЦ рЃўрЃЦрЃюрЃћрЃЉрЃљ:
```json
{
  "web": {
    "client_id": "123456789.apps.googleusercontent.com",
    "client_secret": "GOCSPX-xxxxxxxxxxxxxxxx",
    "redirect_uris": ["..."]
  }
}
```
3. рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ `client_id` рЃЊрЃљ `client_secret` рЃЌрЃЦрЃЋрЃћрЃюрЃА `.env` рЃцрЃљрЃўрЃџрЃерЃў

## рЃюрЃљрЃЉрЃўрЃ»рЃў 7: рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊрЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ (Refresh Token-рЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ)

рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃарЃљрЃф backend рЃЎрЃЮрЃЊрЃА рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃћрЃЉрЃЌ, рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:

рЃћрЃА рЃњрЃљрЃ«рЃАрЃюрЃўрЃА browser-рЃА рЃЊрЃљ рЃЏрЃЮрЃњрЃћрЃЌрЃ«рЃЮрЃЋрЃћрЃЉрЃљрЃЌ:
1. Google-рЃўрЃЌ рЃерЃћрЃАрЃЋрЃџрЃљ
2. FishHunt-рЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃфрЃћрЃЏрЃљ YouTube-рЃќрЃћ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА
3. рЃњрЃљрЃЊрЃљрЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ redirect URI-рЃќрЃћ
4. Refresh token рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ `.env` рЃцрЃљрЃўрЃџрЃерЃў

- РюЁ Test User рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў OAuth consent screen-рЃерЃў
- РюЁ .env рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў
- РюЁ Redirect URIs рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў

## рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў

рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃћрЃбрЃљрЃърЃќрЃћ рЃЊрЃљрЃЋрЃгрЃћрЃарЃЌ:
1. Backend рЃЎрЃЮрЃЊрЃА YouTube-рЃќрЃћ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА
2. Frontend рЃцрЃЮрЃарЃЏрЃљрЃА рЃЋрЃўрЃЊрЃћрЃЮ рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ
3. рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃа integration-рЃА рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃАрЃЌрЃљрЃю

## рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў

Рџа№ИЈ **рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ:**
- рЃљрЃарЃљрЃАрЃЮрЃЊрЃћрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЮрЃЌ Git-рЃерЃў: `youtube-credentials.json`, `youtube-service-account.json`
- `.env` рЃцрЃљрЃўрЃџрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА `.gitignore`-рЃерЃў
- рЃљрЃарЃљрЃАрЃЮрЃЊрЃћрЃА рЃњрЃљрЃљрЃќрЃўрЃљрЃарЃЮрЃЌ Client Secret рЃЊрЃљ Refresh Token

Рџа№ИЈ **Quota Limits:**
- YouTube API-рЃА рЃљрЃЦрЃЋрЃА daily quota: 10,000 units per day
- рЃћрЃарЃЌрЃў video upload РЅѕ 1600 units
- рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃарЃЮрЃЏ рЃЊрЃдрЃћрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ ~6 рЃЋрЃўрЃЊрЃћрЃЮ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЮрЃЌ
- рЃЏрЃћрЃбрЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ quota extension-рЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ

## рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ

рЃЌрЃБ рЃарЃљрЃЏрЃћ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ рЃњрЃљрЃЦрЃЋрЃЌ:
1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ API enabled рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ
2. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ OAuth consent screen test users-рЃерЃў рЃ«рЃљрЃарЃЌ рЃЌрЃБ рЃљрЃарЃљ
3. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ redirect URIs рЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ
4. рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ .env рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў

рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЊрЃљрЃБрЃЉрЃарЃБрЃюрЃЊрЃўрЃЌ рЃљрЃЏ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃА! ­Ъџђ
